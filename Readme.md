# Prediction Regressive Order

This project is a machine learning pipeline for prediction using MLR

### Project Structure

- `data/dataset.csv` — Main dataset containing car listings and their attributes (use-yours from kaggle)
- `model/` — Directory for saving trained models (currently empty).
- `src/` — Source code for data processing and modeling (currently empty).
- `notebooks/playground.ipynb` — Jupyter notebook for experimentation and prototyping.


### Requirements

Install dependencies with:

```bash
pip install -r requirements.txt
```

Main libraries used:
- pandas
- numpy
- scikit-learn
- matplotlib
- seaborn
- joblib
- jupyter

### Usage

1. **Data & Prototyping:**
	- Open `notebooks/playground.ipynb` in Jupyter to get data and build models for interactively.
2. **Model Training:**
	- Implement scripts in `src/` for data preprocessing, training, and evaluation.
3. **Model Saving:**
	- Save trained models to the `model/` directory using `joblib` or similar tools.

### Getting Started

1. Clone the repository.
2. Install the requirements.
3. Explore the notebook or add scripts to `src/` for custom workflows.
